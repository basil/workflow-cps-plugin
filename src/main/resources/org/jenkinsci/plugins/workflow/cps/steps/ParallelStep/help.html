Takes a map from branch names to closures. An optional failFast argument determines whether the step will terminate all branches upon a failure in any other branch. An optional propagateWorst argument determines whether the first error or the worst error (e.g., a failure is worse than an unstable result) will be propagated as the result of the step. 

    parallel firstBranch: {
        // do something
    }, secondBranch: {
        // do something else
    },
    failFast: true|false,
    propagateWorst: true|false
